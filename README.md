# My custom programming configs
As I could likely break my linux installation at any moment, I've archived these here to get back up and running a little faster should (or when) I need them.

## Dependencies

- [**Zed**](https://zed.dev/) - Preferred universal editor

- [**Zoxide**](https://github.com/ajeetdsouza/zoxide) - Upgrade for cd

- [**Starship**](https://starship.rs/) - Terminal prompt rice

- [**Fastfetch**](https://github.com/fastfetch-cli/fastfetch) - Terminal startup rice

- [**Lazygit**](https://github.com/jesseduffield/lazygit) (Optional) - Streamlined TUI Git interface

- [**Rust**](https://rust-lang.org/) (Optional) - One of my prefered langauges. Remove the following line from the [.bashrc](.bashrc) if not wanted:
```bash
. "$HOME/.cargo/env"
```
